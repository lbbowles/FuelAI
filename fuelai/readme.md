## Installation

### Step 1:

Download the repo and link the fuelai to Laravel.

Versions: PHP 8.4, Node 23

### Step 2

After which, open the terminal and run:

npm install

then:

npm run build

### Step 3

Then attach the env file to the database created from the dump. 

### Step 4

Then you should be good to launch the website and register and or sign in. 

As no login is included to get access to the admin page, you must change the role to admin in the database.

### Step 5:

Get your openrouter key from:

https://openrouter.ai/ 

and fill it in the VITE_OPENROUTER_API_KEY slot in the .env file
